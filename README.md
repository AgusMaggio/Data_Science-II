# Data Science II - Primera Preentrega

## Proyecto
Análisis exploratorio sobre la relación entre superficie y población mundial utilizando la API pública REST Countries.

## Objetivo
Evaluar si el tamaño territorial influye en la magnitud demográfica y obtener insights a partir de visualizaciones.

## Fuente de datos
https://restcountries.com/

##Contenido
- Preentrega_Data_Science_II_Maggiore.ipynb (Análisis exploratorio)
- Entrega_Final_DS2_Maggiore.ipynb (Machine Learning - Clasificación)
- countries_raw.csv

##Estado
Proyecto completo. Primera instancia: EDA. Entrega final: Modelos de Machine Learning.
